# Conditional Operations (Java)
```java
1. if Statement

if (condition) {
    // execute if true
}

-----------------------------------

2. if-else

if (condition) {
    // true case
} else {
    // false case
}

-----------------------------------

3. else-if

if (cond1) {
    // case 1
} else if (cond2) {
    // case 2
} else {
    // default
}

-----------------------------------

4. Ternary Operator

result = (condition) ? trueValue : falseValue;

5. switch

switch (var) {
    case value1: // code; break;
    case value2: // code; break;
    default: // code
}

-----------------------------------

Key Rules
- Use == for primitives
- Use .equals() for objects
- && (AND), || (OR) support short-circuiting
